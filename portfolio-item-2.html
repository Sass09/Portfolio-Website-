Google Chrome Extension

Save all your favorite tabs and website links in one place!

This extension is built through Google Extensions to save your current tabs and any links you wanto save in one place. During this project, I learned new things such as inner.HTML, and utilized localStorage to save my tabs. You can click on the SAVE INPUT button to save the link you want to save. You can also click on SAVE TAB button to save the current tab.
