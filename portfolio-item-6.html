The Self Reference Effect

A Computational Memory Research Thesis

The experiment was programmed using HTML and jPsych library with the help of JATOS. The analysis was conducted using R Programming. The main libraries used for the analyses were dplyr and tidyverse There were also some helper function used throughout the code especially for conducting anova. These functions were used to improve the readibilty of the code and to provide descriptions of the variables. An example of one helper function is the get_mean_sem function.

The first graph show the two hypotheses we proposed for our study. The first hypothesis posited that repeating the words will increase memory in semantic and case conditions however the number of words remembered in the self will remain the same.

The second graph shows the results of our experiments 1b and 1c. The results support our first hypothesis as our analysis showed signifcant differences for the case and semantic condition but not for the self-reference condition
